While

Cria um loop. Pode ser colocado uma condição para a execução do loop, e enquanto essa condição for verdadeira o loop continuo.

Do...While.

O "Do..While" inverte o quando a condição será lida pelo programa. Enquanto no While a condição e lida em cima, no Do...While ela e a ultima.

EX: Let ex = 0; While(ex < 3) { alert(ex); ex++; } EX 2: Let ex = 0; Do { alert(ex); ex++; } While(ex < 3);

For

Ele e mais complexo, mas e o mais usado. Ele segue uma ordem de execuçãp:

  • 1 Vá para o começo.
  • 2 Se condição TRUE -> Vá para corpo e depois passo.

Um exemplo basico: For (Começo; Condição; Passo) { //Loop Body }

E possivel usar qualquer variavel ja existente, ou pode criar uma nele mesmo.

Ex: For (let id = 0; id < 3; id++) { alert(id); }

Da pra omitir qualquer uma das partes do For ou ate mesmo não usar nenhuma, mas os dois ponto e virgula são obrigatorios. Um EX: let id = prompt('Digite algo:'); For (;;) { alert(id); }
